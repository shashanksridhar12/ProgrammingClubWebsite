Why Join


Living in the Silicon Valley, programming is an absolutely necessary skill for all high school students to learn.

Coding has numerous applications in today's environment including websites, mobile applications, artifical intelligence, machine learning, and so much more

The FHS Programming Club will expose you to the fundamentals of a variety of programming languages.

One of the best ways to learn how to code is through HACKATHONS, which are fun, interactice coding events where students learn and code an application in 12-24 hours.
